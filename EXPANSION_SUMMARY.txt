1. RETURN TRIP MATCHING (её°гӮҠдҫҝгӮ·гӮ§гӮў) вӯҗ CORE FEATURE
   в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
   в”Ӯ Search Query: Tokyo (35.6762, 139.7674) вҶ’ Kochi (33.5553, 133.5307) в”Ӯ
   в”Ӯ                                                              в”Ӯ
   в”Ӯ Returns:                                                     в”Ӯ
   в”Ӯ  - Normal Trips: Origin вүҲ Tokyo, Destination вүҲ Kochi       в”Ӯ
   в”Ӯ  - Return Trips: Origin вүҲ Kochi, Destination вүҲ Tokyo      в”Ӯ
   в”Ӯ                (Drivers going back can carry cargo)         в”Ӯ
   в”Ӯ                                                              в”Ӯ
   в”Ӯ Algorithm: Haversine Formula                               в”Ӯ
   в”Ӯ  - Distance = 2 * R * asin(sqrt(a))                        в”Ӯ
   в”Ӯ  - Where a = sinВІ(О”lat/2) + cos(lat1)*cos(lat2)*sinВІ(О”lon/2)в”Ӯ
   в”Ӯ  - R = Earth radius (6371 km)                              в”Ӯ
   в”Ӯ  - Radius: Default 50km (configurable)                     в”Ӯ
   в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

================================================================================
                            GETTING STARTED
================================================================================

Step 1: Configure Environment
  export JWT_SECRET="your_very_secure_random_secret_key_here"
  (or update .env file)

Step 2: Run Application
  Option A - Docker: docker-compose up
  Option B - Local:  go run cmd/server/main.go

Step 3: Register User
  curl -X POST http://localhost:8080/api/v1/auth/register \
    -H "Content-Type: application/json" \
    -d '{"email": "driver@example.com", "password": "pass123", ...}'

Step 4: Login and Get Token
  curl -X POST http://localhost:8080/api/v1/auth/login \
    -H "Content-Type: application/json" \
    -d '{"email": "driver@example.com", "password": "pass123"}'

Step 5: Use Token for Requests
  curl -X GET http://localhost:8080/api/v1/trips \
    -H "Authorization: Bearer <JWT_TOKEN>"

See QUICK_START.md for complete testing examples.
