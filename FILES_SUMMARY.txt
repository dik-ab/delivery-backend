KEY FEATURES IMPLEMENTED:
=========================
1. User Authentication
   - Register (driver, shipper)
   - Login with JWT token generation
   - Password hashing with bcrypt
   - 24-hour token expiration

2. Trip Management
   - Create trip (drivers only)
   - Get trips with filters
   - Update trip details
   - Delete trip
   - Search open trips

3. Return Trip Matching (帰り便シェア)
   - Special search endpoint for return trips
   - Haversine formula for geographic matching
   - Configurable search radius (default 50km)
   - Date filtering support
   - Returns normal + return matches separately

4. Cargo Matching
   - Create match request (shippers)
   - Approve/reject matches (drivers)
   - Complete matches upon delivery
   - Cargo weight validation

5. Location Tracking
   - Record GPS location (drivers)
   - Get tracking history
   - Get latest location for map display
   - Timestamp tracking

6. Admin Dashboard
   - Platform statistics
   - User management
   - Trip monitoring
   - Match oversight
   - Role management

7. Role-Based Access Control
   - Driver: create trips, approve matches, track location
   - Shipper: search trips, request matches
   - Admin: view all data, manage roles

RUNNING THE APPLICATION:
========================
Option 1 - Docker Compose:
  docker-compose up

Option 2 - Local:
  go run cmd/server/main.go

Option 3 - Build binary:
  go build -o delivery-api cmd/server/main.go

API BASE URL: http://localhost:8080/api/v1
